# KavyaVaani 🎙️📝

**KavyaVaani** is an AI-powered poetry generator built with Django.  
It allows users to either:
- Write their own prompts to generate poems
- Or simply generate a random poem with a single click  
All poems are also converted into audio using AI text-to-speech (TTS) so they can be heard like a soulful recitation.

---

## 🌟 Features

- 🎨 AI-powered poetry generation
- 🔊 Text-to-speech audio for each poem
- ✍️ User input support for prompts
- 🎧 Simple and clean web interface

---

## 🚀 How to Run

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/KavyaVaani.git
   cd KavyaVaani
2. **Install Requirements**
   ```bash
   pip install -r requirements.txt
3. **Run Server**
   ```bash
   python manage.py runserver
